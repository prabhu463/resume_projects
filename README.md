# Resume Projects

A collection of portfolio projects demonstrating full-stack development, mobile development, and embedded systems expertise.

## Projects

### 1. WORKSPOT BENGALURU
**Full-Stack Booking Platform**

A hyper-local service booking platform built with Django (Backend) and React.js (Frontend).

**Key Features:**
- RESTful API architecture for user authentication, service listings, and booking requests
- JWT-based secure authentication
- PostgreSQL database with optimized queries for concurrent user requests
- Real-time booking management system

**Tech Stack:** Django, Django REST Framework, React.js, PostgreSQL, JWT

[View Project →](./workspot-bengaluru/)

### 2. TASKRABBIT LITE
**Mobile Service App**

A cross-platform mobile application for connecting service providers with customers.

**Key Features:**
- React Native for cross-platform iOS/Android development
- Firebase integration for real-time database
- Real-time task status updates using Firestore listeners
- Secure user authentication with Firebase Auth
- Streamlined UI for seamless mobile experience

**Tech Stack:** React Native, Firebase (Auth, Firestore, Storage), Expo

[View Project →](./taskrabbit-lite/)

---

### 3. Aircraft Tracking and Maintenance System
**Embedded Systems Monitoring Solution**

A real-time monitoring and maintenance tracking solution for aircraft using embedded systems.

**Key Features:**
- Real-time sensor data collection via MQTT
- Automated logging of aircraft performance
- Maintenance scheduling with automated alerts for preventive actions
- Early fault detection through statistical anomaly analysis
- Improved operational safety and efficiency

**Tech Stack:** Python, FastAPI, MQTT, PostgreSQL, WebSockets

[View Project →](./aircraft-tracking-system/)
